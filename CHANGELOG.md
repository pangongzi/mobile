# 更新日志

所有针对 MobileLocator 的显著更改都将记录在此文件中。

本项目遵循 [Semantic Versioning](https://semver.org/spec/v2.0.0.html) 规范。

## [Unreleased]

## [2.1.1] - 2025-02-13

### 优化
- 优化 `getIndexData` 方法性能，使用批量解包减少函数调用
- 完善版本号解析文档，明确说明 842020146 对应 2025年02月
- 优化 README 文档结构和内容

### 修复
- 修正 `parseLocationInfo` 方法返回类型声明为 `?array`

## [2.1.0] - 2025-02-10

### 新增
- 添加性能测试脚本和版本号分析工具
- 完善数据文件版本号解析功能

### 优化
- 性能优化，索引解析效率提升约25%
- 改进错误处理和输入验证

## [2.0.0] - 2025-02-08

### 重大变更
- 重构
- 采用内存映射技术替代文件流读取
- 实现完整的类型声明和现代PHP特性

### 新增
- 单例模式设计
- 二分查找算法优化
- 完整的API文档

## [1.0.0] - 2023-12-01

### 新增
- 初始版本发布
- 基于原作者数据格式的手机号归属地查询功能
- 支持Composer安装和基本查询功能
